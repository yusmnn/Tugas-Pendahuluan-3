Tugas Pendahuluan

1. Jelaskan apa yang dimaksud dengan CSS!
Jawaban:
Cascading Style Sheets (CSS) adalah suatu bahasa stylesheet yang digunakan unutk mengatur tampilan suati dokumen yang ditulis dalam bahasa markUp. Penggunaan yang paling umum dari CSS adalah untuk memformat halaman web yang ditulis dengan HTML dan XHTML. Walaupun demikian, bahasanya sendiri dapat dipergunakan untuk semua jenis dokumen XML termasuk SVG dan XUL.

2. Jelaskan 3 cara pemanggilan CSS serta tuliskan bentuk penggunaannya!
Jawaban:

  • Internal Style Sheet atau Embed Style yaitu berada didalam tag <head> dimana untuk CSS didefinisikan sebagai berikut: 

    • <style type = "text/css">
                    /* Code */ 
                  </style>

  • Inline Style yaitu menjadi atribut dalam tag HTML, atributnya adalah style. Cara pendefinisian property dan nilainya adalah sebagai berikut:

    • <font style = "color: white">
                    Warna tulisan akan menjadi putih 
                  </font>

  • Terdapat di dalam 1 file tertentu. (External Style Sheet atau Embed Style), Pemanggilannya:

    • <link rel = "stylesheet" href = "path/file.css">

3. Jelaskan apa yang dimaksud dengan selector, declaration, property dan value pada CSS!
Jawaban:

  • Selector: digunakan untuk mencari bagian web yang ingin dimanipulasi atau yang ingin di-style. Terdapat 6 macam selector pada CSS yaitu, Selector Tag, Class, ID, Atribut, Universal, dan Pseudo.
  • Declaration: adalah property dan value yang telah berpasangan. Tiap declaration diawal dengan kurung kurawal "{}" dan ditutup dengan titik koma (;).
  • Property: adalah element atau aspek yang ingin kita desain dari selector. Contoh dari property adalah "color" untuk warna, "font-size" untuk ukuran font, "font-family" untuk jenis font, dan sebagainya.
  • Value: adalah pasangan property yang menggambarkan nilai atau hasil akhir dari desain web.

    Contoh value "red", "color", "12" untuk property "font-size".

4. Jelaskan perbedaan selector tag, class dan id pada CSS serta berikan contoh penggunaannya!
Jawaban:

  • Selector Tag: Selektor Tag disebut juga Type Selector. Selektor ini akan memilih elemen berdasarkan nama tag.

    Contohnya pilih semua elemen <p> lalu atur warna teksnya menjadi merah.

    p {
                      color: red;
                    }
  • Selector class: Selektor class adalah selektor yang memilih elemen berdasarkan nama class yang diberikan. Selektor class dibuat dengan tanda titik di depannya.

    Contoh: 

    .container {
                        width: 500px;
                      }
    Selector class bernama .container untuk mengatur lebar di class tersebut.

  • Selector ID: Selektor ID adalah selektor yang memilih elemen berdasarkan nama Id. Selektor Id hampir sama dengan class. Bedanya, ID bersifat unik. Hanya digunakan oleh satu elemen saja.

    Contoh: 

    #contact {
                        align-content: center;
                      }
    Selektor ID ditandai dengan tanda pagar (#) di depannya.
